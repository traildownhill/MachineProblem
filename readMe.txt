HOW TO RUN

REQUIREMENTS:
1. Download and Install any local web server such as XAMPP/WAMP.
2. Download the source code zip file.

INSTALLATION/SETUP:
1. Open your XAMPP/WAMP's Control Panel and start the Apache and MySQL.
2. Extract the downloaded source code zip file.
3. If you are using XAMPP, copy the extracted source code folder and paste it into the XAMPP's "htdocs" directory. And If you are using WAMP, paste it into the "www" directory.
4. Browse the PHPMyAdmin in a browser. i.e. http://localhost/phpmyadmin
5. Create a new database naming "machine_problem".
6. Import the provided SQL file. The file is known as "machine_problem.sql" located inside the sql folder.
7. Browse the MachineProblem in a browser. i.e. "http://localhost/MachineProblem".
